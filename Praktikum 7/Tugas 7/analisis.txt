Pada Praktikum Pemrograman Dasar I kali ini dilakukan 3 percobaan. Pertama,Tugas 1 yaitu membuat program yang mengurutkan sebuah array dalam urutan naik dengan nilai data yang dimasukkan oleh user. Program dimulai dengan meminta pengguna untuk memasukkan Jumlah elemen dalam array. Kemudian, program memasuki perulangan for yang meminta pengguna untuk memasukkan setiap elemen array. Selanjutnya, program memasuki perulangan bersarang yang digunakan untuk mengurutkan array dalam urutan naik.Pada setiap perulangan dari loop luar, program membandingkan setiap elemen array dengan elemen setelahnya. Jika elemen saat ini lebih besar daripada elemen berikutnya, program akan menukar elemen tersebut menggunakan variabel sementara, yaitu a. Setelah perulangan bersarang selesai, program mencetak array yang telah diurutkan.

Kedua, Tugas 2 yaitu membuat program yang menghasilkan semua bilangan prima hingga nilai tertentu yang dimasukkan oleh user dengan menggunakan algoritma sieve of eratosthenes. Program dimulai dengan meminta pengguna memasukkan nilai maksimum bilangan prima yang ingin dihasilkan. Kemudian, program menginisialisasikan array bilangan prima dengan ukuran n+1, dan semua elemennya di-set ke 0. Lalu, program masuk ke dalam perulangan For yang mengulang dari 2 hingga nilai maksimum n yang diberikan. Pada setiap iterasi, program memeriksa apakah bilangan saat ini adalah bilangan prima yang sesuai masih bernilai 0. Jika bilangan saat itu adalah bilangan prima, maka program akan mencetak bilangan tersebut. Selanjutnya, program memasuki perulangan bersarang yang menetapkan semua kelipatan bilangan prima saat ini menjadi 1 dalam array prima. Hal ini dilakukan untuk menandai semua bilangan bukan prima dalam array. Setelah perulangan bertingkat selesai, perulangan luar berlanjut ke bilangan berikutnya dan mengulangi prosesnya hingga semua bilangan prima hingga nilai maksimum n yang diberikan telah dihasilkan.

Ketiga, Tugas 3 yaitu membuat program yang menghitung nilai rata-rata untuk setiap mahasiswa dalam sebuah matriks nilai.
Program mendeklarasikan Sebuah array dua dimensi, dengan ukuran 4x3 untuk menyimpan nilar mahasiswa. Setiap baris mewakili nilai seorang mahasiswa, dan setiap kolom mewakil sebuah mata kuliah. Kemudian, program memasuki perulangan For yang mengulang setiap baris dari array nilai. Pada setiap perulangan, program
menghitung nilai rata-rata dengan menjumlahkan nilai pada setiap kolom dan membaginya dengan jumlah kolom yaitu 3. Kemudian,nilai rata-rata tersebut disimpan ke variabel average dan dicetak dengan menggunakan format tertentu.
